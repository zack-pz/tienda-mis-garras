## API Conventions

- `src/modules` contiene módulos de negocio con arquitectura limpia.
- `src/common` contiene infraestructura transversal compartida.
- NestJS es el framework principal para la mayoría de integraciones.
- Drizzle es el ORM oficial del proyecto.
- Toda API HTTP debe documentarse con Swagger.
- Los endpoints deben devolver códigos HTTP predecibles y consistentes.
